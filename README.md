# De la vallée d'Eska - Site

Installation

1. npm install
2. npm run dev

Import CSV -> JSON

1. placez vos CSV dans /data
2. npm run import-csv

Déploiement sur Vercel

1. Créez un repo GitHub et poussez ce dossier
2. Importez le repo sur Vercel ("New Project -> Import Git")
3. Configurez la variable d'environnement NEXT\_PUBLIC\_ADMIN\_PASS pour l'admin (ex: Eska2025!)
4. Ajoutez vos domaines (delavalleedeska.fr et .com) dans Vercel et suivez les instructions DNS
